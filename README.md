erl_vcron
=========

Given a [calendar:datetime()](http://erlang.org/doc/man/calendar.html#type-datetime)
and a [Vixie Cron-like expression](https://en.wikipedia.org/wiki/Cron),
returns `true` if the expression covers the datetime().
